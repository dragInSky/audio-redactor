# audio-redactor

### Описание
Приложение с графическим интерфейсом для редактирования аудиофайлов в форматах wav и mp3.

### Возможности редактирования:
- [x] разрезка/склейка
- [ ] вырезание фрагмента
- [ ] изменение громкости
- [x] изменение скорости
- [ ] наложение нескольких аудио фрагментов
- [x] reverse звука
